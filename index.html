How It Works

1 ⬇
Download the Launcher

Click your platform above to download the Ironveil Launcher installer. Run it once to install — no configuration needed.

2 ⚒
Sign In to the Forge

Open the launcher and sign in with your Ironveil account. If you don't have one, create it here — access is granted by the dev team when a playtest slot opens.

3 ▶
Install & Play

Once access is granted, the launcher shows an Install button. It downloads the latest build automatically, then lets you launch straight into the game.
